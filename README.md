# ISBN Validator (Python)

This program validates ISBN-10 and ISBN-13 codes.

## Features
- Validates correct ISBN length
- Detects invalid characters
- Supports ISBN-10 (including 'X' check digit)
- Validates ISBN-13 codes
- Handles incorrect input formats

## How to use
Run the program and enter:
ISBN,length

Example:
1530051126,10

## Output
- Valid ISBN Code.
- Invalid ISBN Code.
- Error messages for incorrect input
